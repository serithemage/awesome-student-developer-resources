# awesome-student-developer-resources
학생 개발자를 위한 할인 및 무료 프로그램 모음입니다.

> https://bit.ly/awesome-academy

## 개발 도구
- [Visual Studio for Students](https://visualstudio.microsoft.com/students/) - Microsoft에서 제공하는 학생용 Visual Studio 프로그램입니다. 강력한 통합 개발 환경(IDE)인 Visual Studio를 무료로 사용할 수 있으며, 다양한 플랫폼과 프로그래밍 언어에서 코딩, 디버깅, 테스트를 지원하는 기능과 도구를 제공합니다. 학생들이 소프트웨어 개발 실무 경험을 쌓고, 다양한 기술을 탐색하며, 프로젝트를 구축할 수 있도록 돕습니다.
- [JetBrains 무료 교육용 라이선스](https://www.jetbrains.com/community/education/#students) - JetBrains는 학생들에게 IntelliJ IDEA, PyCharm, PhpStorm 등 전문 소프트웨어 개발 도구를 무료로 제공합니다. 라이선스를 받으려면 학생 신분을 증명해야 합니다.
- [GitHub Student Developer Pack](https://education.github.com/pack#offers) - 학생을 위한 소프트웨어 개발 도구 및 서비스 모음으로, GitHub Pro, JetBrains IDE, Canva 등이 포함됩니다. 13세 이상 학생이 신청할 수 있습니다.
- [Unity Student Plan](https://unity.com/products/unity-student) - Unity는 학생들에게 Unity Pro 게임 개발 플랫폼에 대한 무료 접근권을 제공합니다. 실시간 렌더링, 멀티플레이어 등의 기능이 포함됩니다.
- [Unreal Student Plan](https://www.unrealengine.com/en-US/students) - Epic Games에서 제공하는 프로그램으로, 학생들이 Unreal Engine을 교육 목적으로 무료 사용할 수 있습니다. 게임 개발 학습, 프로젝트 제작, 인터랙티브 경험 구축 등에 활용할 수 있습니다.
- [Docker Personal](https://www.docker.com/products/personal/) - Docker는 Docker Personal 플랜을 통해 학생과 교육자에게 Docker Desktop을 무료로 제공합니다. Docker Desktop, Docker Hub 접근, 컨테이너화 학습 자료가 포함됩니다.

## 클라우드
- [AWS Educate](https://aws.amazon.com/education/awseducate/) - Amazon Web Services(AWS)에서 학생과 교육자에게 클라우드 컴퓨팅과 AWS 서비스를 배울 수 있는 리소스와 도구를 제공합니다. 무료 AWS 크레딧, 학습 경로, 취업 게시판, AWS 자격증 준비 등이 포함됩니다. 학생들이 실무 경험을 쌓고, 기술을 개발하며, 클라우드 컴퓨팅 분야의 잠재적 고용주와 연결될 수 있도록 돕습니다.
- [Microsoft Azure for Students](https://azure.microsoft.com/en-us/free/students/) - 학생들이 Microsoft Azure를 통해 가상 머신, 스토리지, 데이터베이스 등 다양한 클라우드 서비스에 무료로 접근할 수 있습니다. 학생 신분 인증이 필요합니다.
- [Google Cloud for Students](https://cloud.google.com/edu/students) - Google에서 학생들에게 Google Cloud Platform(GCP) 리소스와 서비스에 대한 무료 접근권을 제공합니다. 컴퓨팅, 스토리지, 머신러닝, 데이터 분석 등 클라우드 기술을 탐색하고 실험할 수 있습니다. GCP 실무 경험을 쌓고, 프로젝트를 구축하며, 클라우드 컴퓨팅 기술을 개발할 수 있습니다.

## 협업 도구
- [Atlassian for Education](https://www.atlassian.com/solutions/survey/classroom-license-request) - Atlassian은 교육 기관에 Jira, Confluence, Bitbucket 등 개발 및 협업 도구 제품군에 대한 무료 접근권을 제공합니다. 학생들은 소속 교육 기관의 라이선스를 통해 이러한 도구를 사용할 수 있습니다.

## 디자인 도구
- [Adobe Creative Cloud for Students](https://www.adobe.com/creativecloud/buy/students.html) - Adobe는 인증된 교육 기관에 재학 중인 학생들에게 Photoshop, Illustrator, Premiere Pro 등이 포함된 Creative Cloud 제품군을 할인 또는 무료로 제공합니다.
- [Figma for Education](https://www.figma.com/education/) - Figma와 FigJam은 전문 디자이너, 엔지니어 등이 사용하는 디자인 및 협업 소프트웨어입니다. 학생이나 교사라면 무료로 아이디어를 구상하고, 창작하고, 작업을 공유할 수 있습니다.

## 생산성 도구
- [Notion for Students and Educators](https://www.notion.com/product/notion-for-education) - Notion은 노트 작성, 작업 관리, 위키 생성 등 다양한 기능을 제공하는 종합 생산성 도구입니다. 학생과 교육자에게 무료 Plus 플랜을 제공하며, 학교 이메일로 가입하면 이용할 수 있습니다.

## AI 도구
- [ChatGPT Edu](https://openai.com/chatgpt/education/) - OpenAI의 ChatGPT Edu는 대학 및 고등 교육 기관을 위해 설계되었습니다. 학생, 교직원에게 GPT-4o 접근, 기업 수준의 보안, 높은 메시지 한도, 데이터 분석, 튜터링, 연구 지원 도구를 제공합니다. 기관에서 합리적인 가격으로 캠퍼스 전체에 배포할 수 있습니다.
- [Claude for Education](https://www.anthropic.com/education) - Anthropic의 Claude for Education은 고등 교육을 위한 특화된 Claude 버전을 제공합니다. 가이드 학습을 위한 Learning mode, 학생 프로젝트를 위한 API 크레딧, Northeastern, LSE, Champlain College 등 대학과의 파트너십이 포함됩니다. 기관에서 Anthropic에 연락하여 캠퍼스 전체 접근권을 요청할 수 있습니다.
- [Gemini for Students](https://gemini.google/students/) - Google은 자격을 갖춘 대학생에게 Google One AI Premium(Gemini Advanced 포함)을 최대 12-15개월 무료로 제공합니다. 최신 모델 기반의 Gemini Advanced, 2TB 저장공간, NotebookLM Plus, Deep Research 등의 도구가 포함됩니다. SheerID를 통해 재학 여부를 인증해야 합니다.
