# awesome-student-developer-resources
학생 개발자를 위한 할인 및 무료 프로그램 모음입니다.

> https://bit.ly/awesome-academy

## 개발 도구
- [Visual Studio Community](https://visualstudio.microsoft.com/vs/community/) - Microsoft에서 제공하는 무료 통합 개발 환경(IDE)입니다. 학생, 오픈소스 기여자, 개인 개발자가 무료로 사용할 수 있으며, GitHub Student Developer Pack을 통해 Visual Studio Enterprise 1년 무료 혜택도 제공됩니다.
- [JetBrains 무료 교육용 라이선스](https://www.jetbrains.com/community/education/#students) - JetBrains는 학생들에게 IntelliJ IDEA, PyCharm, PhpStorm 등 전문 소프트웨어 개발 도구를 무료로 제공합니다. 학교 이메일로 인증하면 1년 라이선스를 받을 수 있으며, 재학 중 무제한 갱신이 가능합니다.
- [GitHub Student Developer Pack](https://education.github.com/pack) - 학생을 위한 $100,000 이상 가치의 소프트웨어 개발 도구 및 서비스 모음입니다. GitHub Pro, GitHub Copilot, JetBrains IDE, Canva 등이 포함됩니다. 13세 이상 학생이 신청할 수 있습니다.
- [Unity Student Plan](https://unity.com/products/unity-student) - Unity는 학생들에게 Unity Pro 게임 개발 플랫폼에 대한 무료 접근권을 제공합니다. 실시간 렌더링, 멀티플레이어 등의 기능이 포함됩니다.
- [Unreal Engine for Students](https://www.unrealengine.com/en-US/students) - Epic Games에서 제공하는 프로그램으로, 학생들이 Unreal Engine을 교육 목적으로 무료 사용할 수 있습니다. 게임 개발 학습, 프로젝트 제작, 인터랙티브 경험 구축 등에 활용할 수 있습니다.
- [Docker Desktop](https://www.docker.com/products/docker-desktop/) - Docker Desktop은 개인 사용자, 학생, 소규모 팀(25명 미만)에게 무료로 제공됩니다. 컨테이너 기반 개발 환경 구축, Docker Hub 접근 등을 지원합니다.

## 클라우드
- [AWS Educate](https://aws.amazon.com/education/awseducate/) - Amazon Web Services(AWS)에서 학생과 교육자에게 클라우드 컴퓨팅과 AWS 서비스를 배울 수 있는 리소스와 도구를 제공합니다. AWS 크레딧($100), Starter Account, 학습 경로, 취업 게시판, AWS 자격증 준비 등이 포함됩니다.
- [Microsoft Azure for Students](https://azure.microsoft.com/en-us/free/students/) - 학생들에게 $100 크레딧과 Azure 서비스 12개월 무료 접근을 제공합니다. 가상 머신, 스토리지, 데이터베이스 등 다양한 클라우드 서비스를 이용할 수 있으며, 학교 이메일로 인증합니다.
- [Google Cloud for Students](https://cloud.google.com/edu/students) - Google에서 학생들에게 Google Cloud Platform(GCP) 리소스와 서비스에 대한 무료 접근권을 제공합니다. 크레딧($50~$300), Qwiklabs 무료 실습, 컴퓨팅, 스토리지, 머신러닝, 데이터 분석 등 클라우드 기술을 탐색하고 실험할 수 있습니다.

## 협업 도구
- [Atlassian for Education](https://www.atlassian.com/education) - Atlassian은 교육 기관에 Jira, Confluence, Trello, Bitbucket 등 개발 및 협업 도구 제품군에 대한 무료 접근권을 제공합니다. 학생들은 소속 교육 기관의 라이선스를 통해 이러한 도구를 사용할 수 있습니다.

## 디자인 도구
- [Adobe Creative Cloud for Students](https://www.adobe.com/creativecloud/buy/students.html) - Adobe는 인증된 교육 기관에 재학 중인 학생들에게 Photoshop, Illustrator, Premiere Pro 등이 포함된 Creative Cloud 제품군을 60% 할인된 가격으로 제공합니다.
- [Figma for Education](https://www.figma.com/education/) - Figma와 FigJam은 전문 디자이너, 엔지니어 등이 사용하는 디자인 및 협업 소프트웨어입니다. 학생이나 교사라면 Figma Professional 플랜을 무료로 사용할 수 있으며, 무제한 팀 프로젝트를 지원합니다.

## 생산성 도구
- [Notion for Students and Educators](https://www.notion.so/students) - Notion은 노트 작성, 작업 관리, 위키 생성 등 다양한 기능을 제공하는 종합 생산성 도구입니다. 학생과 교육자에게 무료 Plus 플랜을 제공하며, 학교 이메일로 가입하면 이용할 수 있습니다.

## AI 도구
- [GitHub Copilot Free](https://github.com/features/copilot) - GitHub에서 제공하는 AI 코딩 어시스턴트입니다. 무료 플랜으로 월 2,000회 코드 완성과 50회 채팅을 이용할 수 있으며, GitHub Student Developer Pack 인증 학생은 GitHub Copilot Pro를 무료로 사용할 수 있습니다.
- [ChatGPT Edu](https://openai.com/chatgpt/education/) - OpenAI의 ChatGPT Edu는 대학 및 고등 교육 기관을 위해 설계되었습니다. 학생, 교직원에게 GPT-4o 접근, 기업 수준의 보안, 높은 메시지 한도, 데이터 분석, 튜터링, 연구 지원 도구를 제공합니다. 기관에서 합리적인 가격으로 캠퍼스 전체에 배포할 수 있습니다.
- [Claude for Education](https://www.anthropic.com/education) - Anthropic의 Claude for Education은 고등 교육을 위한 특화된 Claude 버전을 제공합니다. 가이드 학습을 위한 Learning mode, 학생 프로젝트를 위한 API 크레딧, 다수 대학과의 파트너십이 포함됩니다. 기관에서 Anthropic에 연락하여 캠퍼스 전체 접근권을 요청할 수 있습니다.
- [Gemini for Students](https://gemini.google/students/) - Google은 자격을 갖춘 대학생에게 Google One AI Premium(Gemini Advanced 포함)을 최대 12~15개월 무료로 제공합니다. 최신 모델 기반의 Gemini Advanced, 2TB 저장공간, NotebookLM Plus, Deep Research 등의 도구가 포함됩니다. SheerID를 통해 재학 여부를 인증해야 합니다.
- [Cursor Free Plan](https://www.cursor.com/) - AI 기반 코드 에디터로, 무료 플랜에서 월 50회 프리미엄 AI 요청과 다중 파일 편집 기능을 제공합니다. VS Code 기반으로 익숙한 환경에서 AI 코딩을 체험할 수 있습니다.
- [Kiro for students](https://kiro.dev/students/) - AWS에서 제공하는 AI 코딩 어시스턴트입니다. 코드 생성, 디버깅, 최적화를 지원하며, AWS 서비스와의 통합이 강점입니다. 인증된 학교의 학생은 1년간 매달 1,000 크레딧을 무료로 지원합니다. 미국 내 10개 대학을 시작으로 지원하고 있으며 지원 대상 학교가 아니면 Waitlist 에 등록할 수 있습니다.
- [Continue.dev](https://www.continue.dev/) - 오픈소스 AI 코딩 어시스턴트로 VS Code와 JetBrains IDE를 지원합니다. 자체 API 키를 연결하면 다양한 LLM을 무제한으로 사용할 수 있어, 비용 부담 없이 AI 코딩을 활용하고 싶은 학생에게 적합합니다.
